# postgresql
Spotify API 데이터 수집 과정에서 반복적으로 사용되는 파라미터 정보를 적재합니다.<br>
주기적으로 데이터를 자동 백업하여 AWS S3 스토리지에 보관하고 있습니다.<br><br>
